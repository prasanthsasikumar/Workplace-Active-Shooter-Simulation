# Workplace-Active-Shooter-Simulation

Unity-based workplace active shooter simulation with pedestrian dynamics and enemy AI.

## Overview

This project simulates pedestrian movement and crowd behavior during an active shooter scenario. It incorporates multi-Azure-Kinect arrays to reconstruct and analyze pedestrian movement in corridor environments, supporting research into evacuation dynamics and safety training.

Built with Unity and UModeler for rapid 3D prototyping.

## Publications

- **PedestrianDynamics** -- Related work on pedestrian movement reconstruction and crowd simulation. [[Repo]](https://github.com/prasanthsasikumar/PedestrianDynamics)

## Archive Notice

This repository is an archive of a Unity project developed during my work at Massey University (2021-2022), where I designed a multi-Azure-Kinect array to reconstruct pedestrian movement in corridors. Originally hosted on a private Gitea server, it is preserved here for reference and future recreation.

## Requirements

- Unity 2019.x or later
- UModeler (for 3D modeling within Unity)
- EnemyAI asset package
