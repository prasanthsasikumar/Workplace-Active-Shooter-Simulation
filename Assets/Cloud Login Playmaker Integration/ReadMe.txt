-----CloudLogin.Dev Playmaker Integration----

Thank you for purchasing this asset.

Installation Instructions:

1. Install Playmaker
2. Download CloudLogin.Dev from GitHub and place it in your asset folder. https://github.com/mitchmeyer1/cloud-login
3. Extract the CloudLogin.Dev Playmaker Integration Unity Package.

Demo Scene:

1. Be sure to check out the 'Demo' scene included in this package.
2. Note the prefab called 'CloudLoginInitializer'. This comes from the CloudLogin.Dev GitHub directory you installed during installation.
This prefab must be in the first scene you utilize CloudLogin.Dev actions.
